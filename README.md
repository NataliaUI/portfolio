# Your First Website — Portfolio

This is a minimal starter portfolio site.

How to use
- Open `index.html` in your browser (double-click or use a local server).
- Edit `index.html` to add your name, projects, and content.

Optional: serve locally with Python (from this folder):

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Customize
- Replace the placeholder text with your own projects and bio.
- Add images, styles, or a build step as needed.
