# Natalia Portfolio

This project is a personal portfolio website for Natalia, a UX designer and researcher. It presents her professional identity through a polished one-page experience that combines clear messaging, thoughtful layout, and a modern dark visual style.

## What the website is about

The website introduces Natalia as a designer who creates human-centered digital experiences with empathy, strategy, and research. It highlights her focus on clarity, usability, and meaningful product design.

## Main sections

- Hero section with an introduction and positioning statement
- About section describing her background and approach
- Highlights section featuring core strengths such as research and design systems
- Contact section with a direct way to get in touch

## Visual style

The site uses a dark, immersive aesthetic with:
- deep navy and purple gradients
- bold, editorial headlines
- rounded cards and layered surfaces
- subtle glass-like effects and soft shadows
- responsive layout behavior for smaller screens

## Project files

- index.html — main page structure
- styles.css — visual design, layout, and responsive styling
- brief.md — design brief for the site direction

## Technologies

- HTML
- CSS
- Static website with no framework required

## How to view it

Open index.html in a browser, or serve the folder locally with a simple static server for preview.

## Purpose

This portfolio is meant to make a strong first impression, communicate Natalia’s professional value clearly, and invite visitors to connect about design and research opportunities.
